🐅 How to Install This ThemeClick the green Code button at the top right of this GitHub page and select Download ZIP.Extract the downloaded zip folder anywhere on your computer.Open Google Chrome and type chrome://extensions/ in the URL bar.  In the top-right corner of the Extensions page, turn ON the Developer mode toggle switch.  Click the Load unpacked button on the top left.  Select the extracted folder (the parent directory containing manifest.json).Done! Open a new tab to see the Royal Bengal Tiger theme in action.
